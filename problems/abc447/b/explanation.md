# abc447 b 解説

## 問題概要
英小文字のみからなる文字列 `S` が与えられる。`S` の中で出現回数が最も多い文字を
すべて取り除き、残った文字を元の順序のまま出力する(最頻出が複数種類あれば全部削除)。

- 制約: `1 ≤ |S| ≤ 100`
- 例: `mississippi` → `mpp`(`s`と`i`が4回で最多のため両方削除)

## 解法のポイント
1. **26文字の出現回数を数える**
   `vector<int> cnt(26, 0)` を用意し、各文字 `c` について `cnt[c - 'a']++` で
   カウントする。文字コードの引き算 (`c - 'a'`) で `'a'`〜`'z'` を `0`〜`25`
   のインデックスに変換できる。
2. **最大出現回数を求める**
   `*max_element(cnt.begin(), cnt.end())` で26種類のカウントの最大値 `ma` を得る。
3. **最大出現回数の文字だけを除いて出力を組み立てる**
   `S` を先頭から見て、`cnt[c - 'a'] != ma` の文字だけを結果の文字列 `t` に
   追加していく。「最頻出の文字が複数種類あっても、それぞれの出現回数はすべて
   `ma` に等しい」ため、`cnt[...] == ma` で判定するだけで複数種類の最頻出文字を
   まとめて除外できる。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    string s; cin >> s;
    vector<int> cnt(26, 0);
    for (char c : s) cnt[c-'a']++;
    int ma = *max_element(cnt.begin(), cnt.end());
    string t = "";
    for (char c : s){
        if (cnt[c-'a'] != ma) t += c;
    }
    cout << t << endl;
}
```

## 詰まっていた点と学び
- `log.csv` のメモ「文字コードの引き算を使えるようにする」の通り、`c - 'a'` で
  文字を配列のインデックスに変換する発想が最初に出てこなかったのが詰まった原因。
  英小文字26種類の頻度を数える問題では定番のテクニックなので覚えておく。
- 「最頻出の文字を消す」だけなら1種類とは限らない点に注意。`==` ではなく
  `max_element` で得た最大値と等しいかどうかで判定すれば、同率1位が複数あっても
  自然に全部除外できる。
